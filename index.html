Mutually Exclusive Events, Two-way Tables & Venn Diagrams

1. Mutually vs Non-Mutually Exclusive Events

Hint: Mutually exclusive events cannot happen at the same time. Ask yourself: can both events occur simultaneously? If yes, then they are not mutually exclusive.

2. Two-way Table Activity

Hint: Read carefully the categories for rows and columns. Fill in the table by entering the number of individuals in each group. Double check if the sum of rows and columns make logical sense.

3. Venn Diagram Interactive

Steps:
1. Note the total number of people.
2. Look at how many are in Set A and Set B.
3. The overlap number is the count included in both sets simultaneously.
4. Use the formula: Overlap ≤ min(Set A, Set B).
5. Check your answer by comparing with the total and individual sets.
